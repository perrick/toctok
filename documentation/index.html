Documentation | Toctok

Toctok is a sound notifier build upont CouchDB. It will fetch some data from a CouchDB server and based on some rules, it will try to play a sound.

Why create such a tool ?

  • to start experimenting with a new set of tools : CouchDb and Mercurial, in particular
  • to circumvent the need to build a green lamp

How to set up Toctok ?

  1. install a CouchDB server somewhere
  2. start sending PUT requests to the CouchDB server (some examples can be found in /examples/)
  3. fire up your browser at http://localhost/toctok/admin/
    • configure the CouchDB server
    • add some rules based on the requests you're receiving
  4. add a task to your crontab php /path/to/toctok/index.php
  5. done...
